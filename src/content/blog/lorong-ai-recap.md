On March 6th, the ABC partnered with Lorong AI for a hands-on workshop on agentic coding in the terminal.

We covered:

- **Installation & first build** — Getting Claude Code and Codex set up, then building a CRUD app from a single prompt
- **Context & memory** — CLAUDE.md, AGENTS.md, memory systems, compaction, cost awareness
- **Better inputs** — Plan mode, visual inputs, file references, prompt crafting
- **Workflows** — GitHub integration via `gh` CLI, code review, browser MCP, sub-agents, worktrees
- **MCP servers** — What they are, how to install them, examples (Chrome DevTools, Playwright, Supabase, Slack/Jira)
- **Skills** — Reusable agent capabilities: triggers, workflows, definition of done

Participants built and shared CRUD apps during the session, from expense trackers to wine catalogues.
