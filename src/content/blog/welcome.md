The Agentic Builders Collective started in September 2025 as Claude Code Club SG — a small WhatsApp group for developers in Singapore experimenting with Claude Code.

Since then, we've grown to over 80 active members, held five meetups (including one at Google), run hands-on workshops, and expanded well beyond Claude Code into Codex, Cursor, Lovable, MCP servers, and everything else in the rapidly evolving agentic coding landscape.

This website is a permanent home for the resources, projects, and connections that have been shared in our chat. You can browse the [Resources](/resources) page for curated links, check out the [Showcase](/showcase) for what members have built, or add yourself to the [Community](/community) directory.

The site is open source. Want to add a resource, submit a project, or write a post? Open a PR.

And if you haven't yet — [join the WhatsApp chat](https://chat.whatsapp.com/Be8X9lVJQET4JfkKMW5Qxn).
